つくった目的
　対応回数の多い「お客様の声」の業務負担軽減

メイン機能
　問い合わせメール自動作成

サブ機能
　商品検索

始め方
  index.htmlをブラウザにドラッグ＆ドロップするとツールが起動します(chrome推奨)
　「・・・」の設定を開き４つの必要なCSVファイルをインポートしてください
　「マスター読込」を開き「ASマスターフル.csv」をインポートしてください
　準備は完了です！
　
使い方
　お客様の声をデスクトップに保存し開き、CSV形式で保存
　ドラッグアンドドロップ
　検索ボタンでマスター検索
　メールボタンでメーラー起動

さらに便利に
　「・・・」の設定からメール文章の定型登録、メイン４問屋のアドレス登録ができます
　アドレス登録はアップデートでさらに便利にする予定です！

商品検索
　商品名、商品コード、JANコードで検索できます
　商品案内の列順と同じ並びで表示されるのでコピペできます
　縦方向のコピーはできないので改善予定です

追加予定機能
　アドレスのインポート機能
　アドレスのCCおよびBCC登録機能
　履歴管理機能
　検索商品のソート機能
　絞り込み検索機能
